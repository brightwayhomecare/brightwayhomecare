# Brightway Home Care LLC — Project Information

## Project Name
Brightway Home Care LLC — Static Website

## Purpose
Build a clean informational website that helps families and referral sources understand:
- Who Brightway is
- What services are offered
- Who the home supports
- How to contact and schedule a tour

## Business Summary
Brightway Home Care LLC is a licensed Adult Family Home (AFH) in Madison, Wisconsin.

Key highlights:
- Adult Family Home (3–4 bed category)
- Small home setting
- 24/7 support
- Person-centered care

## Target Audience

- Case managers and care coordinators

Secondary:
