## Business Summary
Brightway Home Care LLC is a licensed Adult Family Home (AFH) in Madison, Wisconsin.

**Key Highlights:**
- Adult Family Home (3-4 bed capacity)
- Small, intimate home setting
- 24/7 supervision and support
- Person-centered care approach
- Founded: 2024

## Features Implemented

### Core Features
- Mobile responsive design (all breakpoints)
- Click-to-call and click-to-WhatsApp
- Contact form with EmailJS integration
- Form validation with real-time error messages
- Auto-reply email to form submitters
- Google Maps embed
- Image gallery with lightbox
- FAQ search and filtering

### UI/UX Features
- Dark gradient hero sections
- Glass morphism cards
- Animated background effects (blur, particles, float)
- Hover animations (lift, glow, scale, rotate)
- Wave SVG dividers
- Sticky sidebar navigation (Privacy/Terms)
- Scroll-to-top buttons
- Trust badges and indicators

### SEO Features
- Page-specific metadata
- Open Graph tags
- Semantic HTML structure
- Accessible forms with ARIA labels
- Alt text on all images
