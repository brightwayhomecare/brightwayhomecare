# Brightway Home Care LLC — Website Structure & ASCII Layout

## Goal
Create a simple, static, professional website that:
- Provides clear information about Brightway Home Care LLC
- Builds trust with families and referral sources
- Encourages calls, WhatsApp messages, and contact form submissions

---

## Pages (Simple Static Website)
1. Home `/`
2. About Us `/about`
3. Services `/services`
4. Gallery `/gallery`
5. Contact Us `/contact`

[Home] [About] [Services] [Gallery] [Contact]

Top Right CTA Buttons:
- Call / WhatsApp
- Send Email
